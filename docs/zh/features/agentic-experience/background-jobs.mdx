---
title: 后台执行与定时任务
description: 即使关闭浏览器，Agent 也能继续运行，并把状态回流到 server 协作界面。
---

Poco 支持超出当前浏览器会话的长时间任务执行。这不只影响单个聊天会话，也支撑了 server/channel 中的持久化 agent 协作。

## 亮点

后台执行首先解决的是“人不盯着页面时，任务仍能继续推进”。

- 关闭浏览器后任务仍可继续运行
- 支持定时任务与延迟执行
- 更适合长任务与异步工作流

## 在 server 协作中的表现

当你在频道里 `@agent` 时，Poco 可以复用该 agent 的持久运行时，让执行持续在后台推进。频道消息流会先看到紧凑的 execution placeholder，而更完整的思考、tool call 和运行细节则进入 execution drawer。这样你不需要一直盯着浏览器标签页，也能持续跟进协作进度。
